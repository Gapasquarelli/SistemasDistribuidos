Construção de uma solução MPI para procurar o maior valor em um vetor de numeros aleatórios, mas dessa utilizando MPI_Scatter e MPI_Gather.

Compilação:
$ mpicc main.c -o main

execução:
mpirun -np N ./main

N = número de processos
